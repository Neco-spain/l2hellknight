How to get and compile L2HellKnight?

1. download tortoiseSVN and install.
2. Move to any folder (Me using Desktop)
3. Put the right mouse button and give "SVN Chceckout"
4.. URL of repository is: https://subversion.assembla.com/svn/l2hellknight Chceckout directory is File Locations
5. Now wait for downloading L2HellKnight files Automatic create folder l2hellknight
6. Now move /l2hellknight/trunk/L2HellKnight_Compilator and use build.bat
7. Wait for building Community, core and datapack.
Full manual on http://www.l2hellknight.com
