Инструкции:
Перед свадьбой, прежде всего вы должны обручиться. Для этого просто выделите предполагаемого партнера и введите команду ".engage" без кавычек. Чтобы это подействовало, вы должны быть друг у друга в списке друзей. Для этого введите "/friendinvite ИмяИгрока" без кавычек, вместо ИмяИгрока - имя вашего партнера.
После обручения, вы можете предложить ему обвенчаться. Вы оба должны быть одеты в свадебную одежду и иметь при себе %replace% adena. После свадьбы, командой ".gotolove" без кавычек, вы можете перенестись к вашему супругу. Если вы захотели развестись, введите ".divorce" без кавычек.
