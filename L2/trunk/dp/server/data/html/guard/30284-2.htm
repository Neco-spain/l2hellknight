Sentinel Knight Alberius
Стражи - это духи охранников леса. Когда богиня Einhasad создала эльфов из листьев Дерева Матери, она создала и стражей, чтобы зашишать лес, который будет служить домом для эльфов. Они до вих пор верны этой первой цели и очень враждебно настроены против других рас, которые пытались завоевать из лес. Даже другие эльфы, если они вредят им каким-то образом, сталкиваются с их жестокостью.
