Collector Yumi
Этот район также известен как Ruins of Pavel. Это одна из исследовательских лабороторий Гигантов сохранившаяся еще с тех времен. Я сюда пришла ради исследования Starstones, которые появляются в этом районе. Обитающие здесь големы сделали рискованными прогулки за пределами бараков. Кроме того остатки отвратительных древних экспериментов еще больше прибавляют опасности этим местам.
