Чтобы поднять уровень клана, вам необходимо следующее:
1 уровень - SP : 30,000, adena : 650,000
2 уровень - SP : 150,000, adena : 2,500,000
3 уровень - SP : 500,000, предмет : Blood Mark
4 уровень - SP : 1,400,000, предмет : Alliance Manifesto
5 уровень - SP : 3,500,000, предмет : Seal of Aspiration
6 уровень - репутация клана : 10,000, игроков в клане : более 30
7 уровень - репутация клана : 20,000, игроков в клане : более 80
8 уровень - репутация клана : 40,000, игроков в клане : более 120
Для получения Blood Mark, охотьтесь на Blood Queen внутри Dragon Valley. Чтобы получить Aliance Manifesto, поговорите с Sir Kristof Rodemai из города Giran. Для получения Seal of Aspiration, встретьтесь с Sir Gustav Athebaldt в городе of Oren.
