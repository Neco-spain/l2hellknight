Как рыбачить: Умения для рыбалки
Катушка и Подсечка, умения которые используются для того чтобы рыба схватила наживку.
Как только рыба схватила наживку, она проявляет один из двух типов поведения. Некоторая рыба может попытатся удрать с крючка. Другие сдаются немедленно.
Если рыба извивается и пытается удрать, не пытайтесь тянуть ее. Это опасно для удочки и дает шанс рыбе удрать.
Когда рыба пытается удрать, используйте катушку и дайте рыбе выдохнутся.
Когда рыба выдохлась, потяните удочку на себя и вытащите ее.
Нельзя использовать катушку когда рыба не двигается, или использовать подсечку когда она вырывается, иначе рыба может удрать.
