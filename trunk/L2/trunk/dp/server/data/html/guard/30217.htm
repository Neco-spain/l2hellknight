Sentinel Berros
Да ниспошлет вас мир этого озера отдых. Мое имя Berry, я один из стражей, которые охраняют этот лес и Mother Tree Glade. Когда я был также молод, как и вы, у меня была возможность путешествовать по континенту. Я видел много разных мест, нони одно из них не было столь же красивым как наша Elven village.
