Guard Kosmos
Вы, должно быть, новичок в этих краях! Эти Golems были созданы бесславным, безумным ученым, гномом, Dr. Chaos! Он использовал метеориты, найденные в этой области, чтобы создать ужасаюших Golems, которых использовал, чтобы завладеть всей областью Pavel! Они действительно соответствуют своему ужасаюшему названию! Лично я никогда не видел их, и могу только представлять, какие они ужасные!
