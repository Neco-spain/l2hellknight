Семена делятся на два типа:обычный и альтернативный.Если вы сажаете обычный тип семян,вместо лута с монстра получите эквивалентный ему Урожай. В случае, если вы сажаете альтернативный тип Семян,Вы получите Урожай и лут,с вероятностью примерно 50%. Лорд установил эту систему для того,чтобы игроки могли выбирать предпочтительный для них тип семян для посева. Всего сушествует четыре типа семян:coda,cobol,codran and coba.Доступность каждого из них обусловлена уровнем персонажа.Coda наиболее подходит для фермеров 10-30 уровней,Cobol для 31-48 уровней,Codran для 49-64 уровней,и,наконец,Coba соответствует старшим уровням,от 65 и выше. Очевидно,что разные семяна дают разный урожай.Определенные семена работают лучше с определенными типами монстров.Если монстр слишком слаб,урожай будет скуден.Если напротив,слишком силен - урожай будет поврежден. Секрет удачного Урожая заключается в том,что фермеру необходимо выбрать монстра, приблизительно соответствуюшего ему по силе.Попытки посадить семена в неподходяшего монстра могут повлечь катастрофические последствия. Вы можете приобретать любые семена в любое время,однако Лорд оставляет за собой право приобретать урожай по своему усмотрению.Вам следует все время быть начеку,и быть уверенным что Ваш урожай востребован.
