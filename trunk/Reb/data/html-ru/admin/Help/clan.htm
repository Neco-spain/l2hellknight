--Кланы-- //pledge create [название клана] - создаст клан персонажу по таргету. //pledge setlevel [уровень] - повысит уровень клана на указанный. //pledge resetcreate - уберет штраф за изгнание человека из клана по персонажу по таргету. //pledge resetwait - уберет штраф с персонажа за выход из клана. //pledge addrep [репутация] - добавить репутацию клану по таргету. //pledge setleader - сменит клан лидера клана по таргету на человек в таргете.
