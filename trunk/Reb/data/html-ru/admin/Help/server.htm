//server - покажет страничку сервера //check_actor - проверит характеристики персонажа по таргету. //setvar [название_параметра параметр] - выставит в таблицу сервер_вариаблес заданные значения. //set_ai_interval [время] - устанавливает альтернативный аи_делей указанному мобу по таргету. //spawn2 [ид] - спавнит как и обычный хендлер, но игнорирует запрет на спавн рейдбоссов. //server_shutdown [время] - выключит сервер через определенное время. //server_restart [время] - пошлет сервер на рестарт через указанное время. //server_abort - прекратит отчет рестарта/шат-дауна.
